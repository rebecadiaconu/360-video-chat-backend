# 360-video-chat-backend

WebVR app made using React, [A-Frame](https://aframe.io/) and [Twilio](https://www.twilio.com/). It offers immersive one-to-one videocalls, using Twilio's API for voice and video, all embedded in a 360Â° image rendered with A-Frame. Users can choose different panoramic scenes and ambiental sounds, or upload custom equirectangular images from URL.
